UI был полностью сгенерирован ИИ (Claude code) за исключением лишь мелких правок в Bindings и тексте, которые вносил уже в процессе разработки.
В приложении есть возможность в настройках менять хранилище с Sqlite на Json, но оно не на что не влияет (настройки сохраняются, подгружаются, корректно отображаются, но вот функционала у них нету)
Вместо этого было реализовано сохранение данных в одно и то же время и в Sqlite и в Json.
Такое решение было принято из-за нехватки времени.
